📖 Cara Publikasi ke GitHub Pages (Sekali Setup)

1.Buat repo Public baru di github.com/new
2.Upload file index.html ini ke root repo
3.Upload folder DATA MASTER/ beserta semua file Excel + MASTER_DATA.xlsx
4.Di repo → Settings → Pages → Branch: main → / (root) → Save
5.Isi username & repo di form atas → Simpan. Dashboard langsung sync otomatis!
↻Update data tiap bulan: cukup upload file Excel baru ke folder DATA MASTER di GitHub
